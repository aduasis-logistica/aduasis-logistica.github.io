# Manejo de errores



### Inconsistent Module State

<img src="/img/portal/errores/inconsistentModuleState.png" />

Para solucionar este error es necesario ir el directorio en donde se encuentra instalado el servidor de glassfish.

Después accesar a la siguiente ruta dentro de la carpeta del servidor "\glassfish\domains\domain1\generated" una vez dentro se debera borrar todo el contenido de la carpeta y volver a arrancar el proyecto.
