# Instalación

## Requerimientos

### Desarrollo móvil

- [Dart 2.19.0](https://storage.googleapis.com/dart-archive/channels/stable/release/2.19.0/sdk/dartsdk-windows-x64-release.zip) - Lenguaje de programación.
- [Flutter 3.7.0](https://storage.googleapis.com/flutter_infra_release/releases/stable/windows/flutter_windows_3.7.0-stable.zip) - Framework de dart para desarrollo multiplataforma móvil.
- [Visual studio](https://visualstudio.microsoft.com/) - Editor de código con extensiones para desarrollo en flutter.
- [Android studio](https://developer.android.com/studio?gclid=CjwKCAiAqaWdBhAvEiwAGAQltoPFh8R9JB326nJUJCzYu2Mi2Qrzcjm5R2APhZCXXUQLy6tgMv3wRBoCCLoQAvD_BwE&gclsrc=aw.ds) - IDE para desarrollo móvil en flutter.

### Desarrollo backend

- [.Net Core 5](https://dotnet.microsoft.com/en-us/download/dotnet/5.0) - Framework para desarrollo de la API.
- [Visual studio code](https://code.visualstudio.com/) - IDE para trabajar con la API.
- [SQL Server (Descargar version developer)](https://www.microsoft.com/es-mx/sql-server/sql-server-downloads) - Motor de base de datos.
- [SQL Server management studio](https://learn.microsoft.com/en-us/sql/ssms/download-sql-server-management-studio-ssms?view=sql-server-ver16) - Gestor de base de datos.
- [Banana cake pop](https://chillicream.com/docs/bananacakepop) - Permite crear peticiones a API tipo GraphQL de forma sencilla.

### Versionador

- [Git](https://git-scm.com/) - Gestor de versiones.
- [Github Desktop](https://desktop.github.com/) - Interactua con git de forma gráfica y sencilla.
- [Repositorio : Flutter APP](https://github.com/fegarza/aduasis_yms) - Frontend de la aplicación.
- [Repositorio : GraphQL API](https://github.com/fegarza/AduasisAPI) - Backend de la aplicación.
